# 19. Roadmap de Desenvolvimento

## Fase 1 — Documentação

- Criar documentos principais.
- Criar diagramas.
- Definir escopo do MVP.
- Definir fluxo de pedido e encomenda.

## Fase 2 — Protótipo visual

- Definir layout da home.
- Definir cards de produtos.
- Definir carrinho.
- Definir formulário de encomenda.

## Fase 3 — Frontend MVP

- Criar projeto React + Vite.
- Criar estrutura de pastas.
- Criar páginas principais.
- Criar componentes reutilizáveis.

## Fase 4 — Cardápio e carrinho

- Criar dados dos produtos.
- Listar categorias.
- Adicionar ao carrinho.
- Calcular total.

## Fase 5 — WhatsApp

- Gerar mensagem de pedido.
- Gerar mensagem de encomenda.
- Abrir WhatsApp com texto formatado.

## Fase 6 — Publicação

- Configurar deploy.
- Configurar domínio.
- Testar em celulares.
- Validar com a pastelaria.

## Fase 7 — Painel futuro

- Criar backend.
- Criar banco.
- Criar login.
- Criar gestão de produtos, pedidos e encomendas.
