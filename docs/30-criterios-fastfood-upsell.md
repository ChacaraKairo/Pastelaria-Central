# 30. Critérios de Aceite da Experiência Fast-Food

## Objetivo

Definir critérios de aceite para validar se o sistema transmite sensação de compra em fast-food e incentiva pedidos mais completos.

## Categorias

- Salgados devem aparecer antes das demais categorias.
- Bebidas devem aparecer logo depois de salgados.
- Combos devem ter destaque visual.
- Encomendas, massa de pastel e congelados devem ficar separados do pedido rápido quando forem produtos sob orçamento.

## Home

- A Home deve ter chamadas comerciais, não apenas textos institucionais.
- A Home deve destacar “Mais pedidos”.
- A Home deve destacar “Salgados para agora”.
- A Home deve destacar “Bebidas geladas”.
- A Home deve destacar “Combos da casa”.
- A Home deve destacar “Encomendas e congelados”.

## Produto

- Produtos principais devem permitir sugestão de bebidas.
- Produtos principais devem permitir sugestão de combos.
- Produtos de encomenda devem direcionar para orçamento.

## Carrinho

- O carrinho deve exibir seção “Complete seu pedido”.
- Se houver salgado ou pastel sem bebida, o carrinho deve sugerir bebidas.
- Se houver bebida sem salgado ou pastel, o carrinho deve sugerir produtos principais.
- Se houver itens individuais sem combo, o carrinho deve sugerir combos.

## WhatsApp

- O pedido via WhatsApp deve continuar funcionando.
- A mensagem deve incluir todos os itens adicionados, inclusive bebidas e combos sugeridos.
- A sugestão de itens não deve bloquear o envio do pedido.

## Encomenda

- Produtos de massa de pastel, congelados e encomendas devem ter tratamento separado quando forem sob orçamento.
- A mensagem de encomenda deve deixar claro que é solicitação de orçamento.

## Experiência esperada

O sistema deve fazer o cliente sentir que está comprando em um ambiente moderno, guiado e comercial, semelhante a uma rede de fast-food ou aplicativo de delivery.
