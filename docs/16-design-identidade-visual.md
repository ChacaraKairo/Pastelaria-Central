# 16. Design e Identidade Visual

## Direção visual

O site deve parecer uma vitrine de fast-food, não apenas um site institucional.

## Objetivo visual

Fazer o usuário sentir vontade de pedir.

## Referências

- McDonald's.
- Burger King.
- Cardápios digitais modernos.
- Aplicativos de delivery.

## Características desejadas

- Fotos grandes de comida.
- Botões fortes e visíveis.
- Destaque para produtos e combos.
- Pouco texto no topo.
- Layout mobile first.
- Carrinho sempre fácil de acessar.

## Elementos principais

- Hero com imagem chamativa.
- Cards de produtos.
- Selos de destaque: mais vendido, promoção, sob encomenda.
- Botão “Pedir agora”.
- Botão “Fazer encomenda”.

## Tom de comunicação

Direto, apetitoso e comercial.

Exemplos:

- Peça seu pastel agora.
- Escolha, monte e envie pelo WhatsApp.
- Encomende massas e congelados para sua cidade.
- Parada obrigatória na Rota Bahia-Minas.
