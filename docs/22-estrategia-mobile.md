# 22. Estratégia Mobile

## Objetivo

Transformar o MVP 1 em uma experiência web mobile-first com aparência de app de pedidos.

## Direção

O cliente deve abrir o site pelo celular, ver produtos com fotos, navegar por categorias, adicionar ao carrinho e finalizar pelo WhatsApp com poucos passos.

## Princípios

- Priorizar pedido rápido.
- Manter navegação inferior fixa no mobile.
- Exibir cardápio com busca e categorias horizontais.
- Usar lista de produtos compacta no celular.
- Manter encomendas separadas do pedido comum.
- Preparar evolução futura para PWA, sem implementar app nativo neste momento.

## Fora do escopo

- React Native.
- App nativo.
- Backend.
- Banco de dados.
- Login.
- Pagamento online.
- Painel administrativo.
