# 03. Escopo do MVP

## Objetivo do MVP

Criar a primeira versão funcional e vendável do site da Pastelaria Central, com foco em apresentação visual, cardápio, pedido rápido e encomendas.

## Entra no MVP 1

- Página inicial estilo fast-food.
- Cardápio digital.
- Filtro por categorias.
- Produtos com nome, descrição, preço e imagem.
- Carrinho local.
- Formulário de finalização de pedido.
- Pedido via WhatsApp.
- Formulário de encomenda via WhatsApp.
- Página Café do Ciclista.
- Página Sobre.
- Página Contato e localização.
- Layout responsivo.
- Conteúdo inicial preparado para SEO local.

## Não entra no MVP 1

- Login administrativo.
- Banco de dados em produção.
- Pagamento online.
- Impressão automática.
- Estoque.
- Relatórios.
- Cupons.
- Fidelidade.
- Integração com iFood.

## MVP 2 planejado

- Backend.
- Banco de dados.
- Painel administrativo.
- Gestão de produtos.
- Gestão de pedidos.
- Gestão de encomendas.
- Status de pedidos e encomendas.

## Critério para o MVP estar pronto

O cliente deve conseguir abrir o site pelo celular, escolher produtos, montar pedido, preencher dados e enviar tudo formatado para o WhatsApp da pastelaria.
