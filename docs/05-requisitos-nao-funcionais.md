# 05. Requisitos Não Funcionais

## Responsividade

- RNF001 — O site deve funcionar bem em celulares, tablets e computadores.
- RNF002 — O layout deve priorizar uso em celular.

## Desempenho

- RNF003 — A página inicial deve carregar rapidamente.
- RNF004 — Imagens devem ser otimizadas para web.
- RNF005 — O carrinho deve responder sem lentidão perceptível.

## Usabilidade

- RNF006 — O botão de pedido deve estar sempre fácil de encontrar.
- RNF007 — O processo de pedido deve exigir poucos passos.
- RNF008 — O cliente deve entender claramente a diferença entre pedido rápido e encomenda.

## Segurança

- RNF009 — O MVP não deve armazenar dados sensíveis em banco de dados.
- RNF010 — Quando houver backend, rotas administrativas devem exigir autenticação.

## Privacidade

- RNF011 — O site deve informar que os dados preenchidos serão enviados pelo WhatsApp.
- RNF012 — Se houver coleta futura de dados, o site deve ter política de privacidade.

## Manutenibilidade

- RNF013 — Produtos devem ficar separados em arquivo de dados no MVP.
- RNF014 — Componentes visuais devem ser reutilizáveis.
- RNF015 — A estrutura deve permitir migração futura para backend.

## SEO local

- RNF016 — O site deve conter termos relacionados a Novo Cruzeiro/MG.
- RNF017 — O site deve conter informações claras de contato e localização.
- RNF018 — As páginas devem ter títulos e descrições adequadas.
