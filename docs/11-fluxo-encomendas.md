# 11. Fluxo de Encomendas

## Objetivo

Permitir que clientes solicitem orçamento para massa de pastel, salgados congelados, grandes quantidades, eventos e pedidos para outras cidades.

## Diferença entre pedido rápido e encomenda

Pedido rápido é para consumo imediato ou no mesmo dia.

Encomenda é para produtos com planejamento, quantidade maior, data futura ou entrega para outra cidade.

## Tipos de encomenda

- Massa de pastel.
- Salgados congelados.
- Cento de salgados.
- Pedido para evento.
- Pedido para outra cidade.
- Combo para festa.

## Etapas

1. Cliente acessa a página Encomendas.
2. Escolhe o tipo de encomenda.
3. Informa produto desejado.
4. Informa quantidade.
5. Informa cidade.
6. Informa data desejada.
7. Escolhe retirada ou entrega.
8. Adiciona observações.
9. Envia solicitação pelo WhatsApp.
10. Pastelaria confirma disponibilidade, prazo e valor.

## Dados obrigatórios

- Nome.
- Telefone.
- Tipo de encomenda.
- Quantidade.
- Cidade.
- Data desejada.

## Modelo de mensagem

```text
Olá, Pastelaria Central! Quero solicitar uma encomenda.

Nome: [nome]
Telefone: [telefone]
Tipo de encomenda: [tipo]
Produto: [produto]
Quantidade: [quantidade]
Cidade: [cidade]
Data desejada: [data]
Retirada ou entrega: [tipo]
Observações: [observações]
```

## Regra importante

A encomenda será tratada inicialmente como solicitação de orçamento, não como pedido confirmado.
