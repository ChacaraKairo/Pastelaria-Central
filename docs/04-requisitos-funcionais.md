# 04. Requisitos Funcionais

## Site público

- RF001 — O sistema deve exibir uma página inicial com chamadas para pedido e encomenda.
- RF002 — O sistema deve exibir informações básicas da Pastelaria Central.
- RF003 — O sistema deve exibir uma seção para o Café do Ciclista.
- RF004 — O sistema deve exibir endereço, contato, Instagram e mapa.

## Cardápio

- RF005 — O sistema deve listar produtos por categoria.
- RF006 — O sistema deve exibir nome, descrição, preço, imagem e disponibilidade do produto.
- RF007 — O sistema deve permitir filtrar produtos por categoria.
- RF008 — O sistema deve destacar produtos promocionais ou mais vendidos.

## Carrinho

- RF009 — O cliente deve conseguir adicionar produtos ao carrinho.
- RF010 — O cliente deve conseguir remover produtos do carrinho.
- RF011 — O cliente deve conseguir alterar a quantidade de um item.
- RF012 — O sistema deve calcular subtotal e total do pedido.

## Pedido rápido

- RF013 — O cliente deve informar nome e telefone.
- RF014 — O cliente deve escolher retirada ou entrega.
- RF015 — O cliente deve informar endereço quando escolher entrega.
- RF016 — O cliente deve escolher forma de pagamento.
- RF017 — O cliente deve adicionar observações ao pedido.
- RF018 — O sistema deve gerar uma mensagem formatada para WhatsApp.

## Encomendas

- RF019 — O cliente deve conseguir solicitar encomenda.
- RF020 — O cliente deve escolher tipo de encomenda.
- RF021 — O cliente deve informar quantidade.
- RF022 — O cliente deve informar cidade.
- RF023 — O cliente deve informar data desejada.
- RF024 — O sistema deve gerar mensagem de solicitação de orçamento pelo WhatsApp.

## Painel futuro

- RF025 — O administrador poderá cadastrar produtos.
- RF026 — O administrador poderá editar preços.
- RF027 — O administrador poderá ativar ou desativar produtos.
- RF028 — O administrador poderá acompanhar pedidos.
- RF029 — O administrador poderá acompanhar encomendas.
