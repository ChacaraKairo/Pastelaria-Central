# Pastelaria Central

Projeto para criação de um site vendedor e sistema de pedidos para a Pastelaria Central de Novo Cruzeiro/MG.

## Objetivo

Criar uma plataforma digital com foco em conversão, inspirada em experiências de fast-food, para:

- apresentar a Pastelaria Central;
- exibir cardápio digital;
- receber pedidos rápidos;
- receber solicitações de encomendas;
- destacar produtos como pastéis, salgados, massa de pastel e congelados;
- valorizar o Café do Ciclista e a relação com a Rota Bahia-Minas.

## Estratégia do projeto

O projeto será desenvolvido em fases.

### MVP 1

Site público com:

- página inicial estilo fast-food;
- cardápio digital;
- carrinho;
- pedido via WhatsApp;
- formulário de encomenda via WhatsApp;
- página do Café do Ciclista;
- contato e localização.

### MVP 2

Sistema administrativo com:

- login;
- cadastro de produtos;
- cadastro de categorias;
- gestão de pedidos;
- gestão de encomendas;
- painel de produção/cozinha.

### MVP 3

Evoluções futuras:

- banco de dados em produção;
- pagamentos online;
- agenda de encomendas;
- relatórios;
- cupons;
- fidelidade;
- estoque básico.
